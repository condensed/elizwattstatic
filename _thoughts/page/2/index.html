⚠️ Missing Page - Export Issue

This page wasn't included in the Simply Static export

The "Thoughts" section pagination pages (page 2 and 3) were not exported by Simply Static. This is a common issue with WordPress static site generators when they don't crawl all pagination links.

What happened?

The Simply Static plugin created a link to /thoughts/page/2/ on the main Thoughts page, but it didn't actually export the content for this URL. This results in a 404 error.

Solutions:

  1. Re-export from WordPress: In the Simply Static settings, ensure that pagination links are being crawled and exported.
  2. Manually create the pages: Copy the content structure from /thoughts/index.html and adjust the pagination.
  3. Remove pagination: Edit /thoughts/index.html to remove the pagination links, or increase posts-per-page to show all content on one page.
